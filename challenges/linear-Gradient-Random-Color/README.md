Hint:  
- `linear-gradient`: 선형그라이데이션을 만드는 css함수
- `Math.floor()`: 주어진 숫자의 소수점 이하를 내림해서 반환 e.g., 1.34 > 1
- `Math.random()`: 0에서 1보다 작은 범위의 난수 반환 e.g., 0.232 0.9232
- `length`: 배열의 길이 반환  e.g. clothing.length
